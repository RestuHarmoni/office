Delete Safety Rule

System Admin hanya delete data operasi project seperti projects, tasks, workflow, logs, outputs dan QA record yang berkaitan dengan projectId. Core system seperti departments, sop, templates, prompts dan knowledge-base tidak disentuh.

Allowed delete: projects/{projectId}, tasks where projectId, workflow where projectId, outputs where projectId, qa_reviews where projectId/outputId, deliveries/final_deliveries where projectId/outputId, workflow_logs where projectId, notifications where projectId.
